Exploring Computer Science

This is an inclusive class for all students and is intended to provide an avenue where students can be creative while learning technologies that shape our world. The course inspires students as they design their own websites, code games, stories and scenes, and design and build apps that solve problems. In our HTML unit, students use HTML and CSS to create personal websites. When learning coding and animation, JavaScript is used to create interactive scenes and games. Students build an understanding of user based design and design mobile apps that solve real world problems. This is a beginner class with no prior experience required and is intended to inspire all students to be creators with technology!
