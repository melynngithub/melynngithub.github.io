CTE Computer Science Foundations

This is an inclusive class for all students and is intended to provide an avenue where students can be creative while learning technologies that shape our world. The primary focus of the course is a deep dive into the fundamentals of programming concepts and teaches text-based coding with introductory Python (new 23-24). This course also includes a brief introduction to physical computing as students learn to code the Adafruit Circuit Playground. Topics in the class will also include problem solving, user-centered design, and trending technology topics such as artificial intelligence as time allows. This is a beginner class with no prior experience required and is intended to inspire all students to be creators with technology!
