Multimedia 2 - Video Game Design

This semester course teaches students the fundamentals of game design by using Unity's game engine. By the end of this course, students will understand the design planning process, be knowledgeable of industry related careers, and be able to navigate the Unity environment in order to create their own 2D and 3D games. Topics also include learning about the game design industry, different roles involved in game design, and ethical issues in the field.
